# bot-p2pchat

A simple P2P chat based on https://dev.to/carloslfu/make-a-p2p-connection-in-10-minutes-4b32

## Usage

```
Usage: chat [options] <message>

  Send a message to everyone running this command

  Options:

    --help  output usage information
```

## Config (config.yaml)

```yaml
p2pchat:
  username: john_doe
```
